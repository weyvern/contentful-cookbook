# Cookbook blog with Contentful

## Clone

```bash
git clone https://github.com/weyvern/contentful-cookbook.git
```

## Install

```bash
npm install
```

## Contentful key

- Create your .env file and add:

* REACT_APP_CONTENTFUL_SPACE_ID
* REACT_APP_CONTENTFUL_ACCESS_TOKEN

## Start dev server

```bash
npm start
```
